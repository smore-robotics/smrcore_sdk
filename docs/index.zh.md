# smrcore_sdk

`smrcore_sdk` 是 Smartmore Robot SDK 的公开文档与示例仓库。

## 开始阅读

- [快速开始](getting-started.md)
- [C++ 示例](examples/index.md)
- [Python 示例](python/examples/index.md)
- [API 参考](api/index.md)

## 仓库内容

- 支持平台的 C++ SDK Release 压缩包与 Python wheel。
- `examples/` 中的 C++ 示例与 `examples_py/` 中的 Python 示例。
- 下载最新 C++ SDK 和构建示例的脚本。
- 从 C++ SDK 头文件生成的 API 参考。

该仓库不包含 SDK 源码。用户从 GitHub Releases 下载带版本号的 SDK 制品，并基于
这些制品构建自己的应用。
